# Cybersecurity Basics 

## What is Cybersecurity?
Cybersecurity is the practice of protecting systems, networks, and data from digital attacks.

## Why Cybersecurity is Important
- Protects personal and financial data
- Prevents cybercrime and fraud
- Keeps online services safe

## Types of Hackers
- White Hat: Ethical security professionals
- Black Hat: Cyber criminals
- Grey Hat: Between ethical and illegal

## Areas of Cybersecurity
- Network Security
- Application Security
- Information Security
- Ethical Hacking
- Digital Forensics
